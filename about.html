QuickFIX/Go is a FIX Protocol implementation for the Go programming language.

QuickFIX/Go is 100% free and open source with a liberal license.

Installation

Installing, or upgrading QuickFIXGo is quite simple (assuming you've already got Go installed)

go get github.com/quickfixgo/quickfix
